# 验收清单

## 功能
- [ ] 注册/登录/鉴权（教师/学生/管理员）
- [ ] 绑定/解绑流转（教师发起/学生审批/学生发起/教师审批）
- [ ] 预约创建（小/大教室、跨天）/冲突/容量/取消
- [ ] 周视图与便捷接口（系统总览、教室详情）
- [ ] 学生视图（本周/历史/筛选）
- [ ] 配置读取/修改（容量/周起始/时区/科目）
- [ ] 报表导出（CSV/PDF）与留存策略

## 权限与安全
- [ ] JWT 401/角色 403 覆盖
- [ ] 输入校验 422（日期/时段/科目/ID/slots）
- [ ] 速率限制生效（auth/预约）
- [ ] CORS 设置正确

## 并发与一致性
- [ ] 小教室并发：仅 1 成功，多数 409
- [ ] 大教室并发：合计不超过 capacity
- [ ] 取消与创建交织：Occupancy 一致
